# blog
简单的小博客

## 目录结构
│  BlogApplication.java
├─config                       配置信息类
├─consist                      常量接口类
├─controller                   路由控制器
│      TagController.java
├─domain                       实体类（bean）
├─repository                   数据访问层（dao）
├─service                      数据访问层
│  └─impl                     数据服务的实现接口
├─utils                        工具类
└─vo                           数据传输类
